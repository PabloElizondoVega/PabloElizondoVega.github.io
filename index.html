Software para Escritores: "Constructor de Escenas"

"Constructor de Escenas" es un pequeño proyecto de software pensado originalmente para tener un procesador de texto más personalizado a la hora de crear escenas para novelas. La ventaja del software radica en que no es un software generalista, siendo una buena alternativa a los procesadores de texto habituales al tener una guía de construcción de escenas incorporada.
